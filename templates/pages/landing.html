Fonctionnalités Clés

Découvrez les capacités puissantes de notre solution MPI

Identifiant Patient Unique (UPI)

Attribution d'un UUID unique à chaque patient pour une identification sans ambiguïté à travers tout le système de santé.

Interopérabilité

Connectez facilement différentes plateformes (Laboratoire, Surveillance, Hospitalisation) via des API sécurisées.

Sécurité Renforcée

Clés API uniques par plateforme, contrôle d'accès granulaire et audit complet des activités.

Recherche Avancée

Trouvez rapidement les patients grâce à des critères multiples (nom, prénom, date de naissance, contact).

Mises à Jour en Temps Réel

Webhooks pour notifier les systèmes connectés des modifications importantes des données patients.

Tableau de Bord Complet

Visualisez les statistiques d'utilisation et suivez les performances du système grâce à des rapports détaillés.

Intégration Technique

Connectez facilement vos systèmes existants

API RESTful Sécurisée

Notre API moderne permet une intégration aisée avec tous vos systèmes :

POST /api/patients/ 
{
    "nom": "Kouakou",
    "prenoms": "Jean Paul",
    "date_naissance": "1985-05-15",
    "contact": "+2250701234567"
}
GET /api/patients/?search=Kouakou 
{
    "count": 1,
    "results": [
        {
            "upi": "a1b2c3d4-e5f6-g7h8-i9j0-k1l2m3n4o5p6",
            "nom": "Kouakou",
            "prenoms": "Jean Paul",
            ...
        }
    ]
}

Documentation API complète disponible après connexion.
